## Features
- **Lift Levels**
- **Lift Sounds**
- **Return Lift Button**
- **Handles the static lift prop being removed from the map**
---

## Dependencies

| Resource | Notes |
|---|---|
| [ox_lib](https://github.com/overextended/ox_lib) | UI notifications and animations |
| [ox_target](https://github.com/overextended/ox_target) | Drone interaction targeting |
---

## Installation

### 1. Add the resource

Drop the `nzkfc_conlift` folder into your `resources` directory (or in whatever subfolder you want) and add the following to your `server.cfg`:

```
ensure nzkfc_conlift
```

## Controls

Simply use Ox Target to target the lift cab or the control boxes.

---

## Configuration

All settings are in `config.lua`.
